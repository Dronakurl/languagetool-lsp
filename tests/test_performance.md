# Performance Test

Dies ist ein Test für die Performance des LSP bei schnellem Tippen.

Der Text sollte sofort beim Laden geprüft werden und während des Tippens sollten neue Fehler erscheinen (auch wenn LanguageTool etwas langsamer ist).

Hier sind einige Testwörter:
- ihc (sollte ich sein)
- Kanl (sollte Kann oder Karl sein)
- möchte (korrekt)
- falsch (korrekt)

Beim schnellen Tippen sollten die Diagnostics mit minimaler Verzögerung erscheinen.
